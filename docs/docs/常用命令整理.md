# 常用命令整理

## 1. Bash

```bash
arch 显示机器的处理器架构
apt 安装包管理工具
apt autoremove 删除为了满足其他软件包的依赖而安装的，但现在不再需要的软件包
apt install 安装软件包
apt update 更新包信息
apt upgrade 更新包
bc 任意精度计算器语言
cal 显示日历表
cat 用于链接文件并打印到标准输出设备上
cat /proc/cpuinfo 显示CPU info的信息
cat /proc/interrupts 显示中断
cat /proc/meminfo 校验内存使用
cat /proc/swaps 显示哪些swap被使用
cat /proc/version 显示内核版本
cat /proc/net/dev 显示网络适配器及统计
cat /proc/mounts 显示已加载的文件系统
cd 用于切换当前工作目录至dirName(进入个人的主目录)
cd .. 返回上级目录
cd - 返回上次所在的目录
clock 用于调整RTC时间（RTC是电脑内建的硬件时间）
cp
date 显示系统日期
df
df -h 显示已经挂载的分区列表
dmesg 用于显示开机信息
dmidecode -q 显示硬件系统部件
env 显示系统中已存在的环境变量
find 查找文件
git
h
i
j
kill
logout 注销
ls
lspci -tv 罗列PCI设备
lsusb -tv 罗列USB设备
mv
nohup
o
pwd 显示工作路径
q
reboot 重启
shutdown now 关机
shutdown -r now 重启
ssh
scp
tree 以树型列出当前目录下的文件和文件夹
uname -m 显示机器的处理器架构
uname -r 显示正在使用的内核版本
v
w 显示日前登入系统的用户信息
whoami
x
y
zip
```
