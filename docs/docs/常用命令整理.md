# 常用命令整理

## 1. Bash

```bash
alias 给命令定义别名(查看别名)
arch 显示机器的处理器架构
apt 安装包管理工具
apt autoremove 删除为了满足其他软件包的依赖而安装的，但现在不再需要的软件包
apt install 安装软件包
apt update 更新包信息
apt upgrade 更新包
bc 任意精度计算器语言
cal 显示日历表
cat 用于链接文件并打印到标准输出设备上
cat /proc/cpuinfo 显示CPU info的信息
cat /proc/interrupts 显示中断
cat /proc/meminfo 校验内存使用
cat /proc/swaps 显示哪些swap被使用
cat /proc/version 显示内核版本
cat /proc/net/dev 显示网络适配器及统计
cat /proc/mounts 显示已加载的文件系统
cd 用于切换当前工作目录至dirName(进入个人的主目录)
cd .. 返回上级目录
cd - 返回上次所在的目录
clock 用于调整RTC时间（RTC是电脑内建的硬件时间）
cp 主要用于复制文件或目录
date 显示系统日期
df 检查文件系统的磁盘占用情况
df -h 显示已经挂载的分区列表
dmesg 用于显示开机信息
dmidecode -q 显示硬件系统部件
docker 容器
du
env 显示系统中已存在的环境变量
echo 字符串的输出
file
find 查找文件
gcc Linux下默认的C/C++编译器
git 版本控制系统
grep 文本搜索工具（可使用正则表达式）
history
ip addr 查看ip地址
iptables 包过滤防火墙
java
jobs
kill 用于删除执行中的程序或工作
logout 注销
ls
lspci -tv 罗列PCI设备
lsusb -tv 罗列USB设备
mv
nohup
opencv_version
paste 合并拼接文件
pwd 显示工作路径
q
read
reboot 重启
sed
shutdown now 关机
shutdown -r now 重启
sl 跑火车
ssh
scp
tree 以树型列出当前目录下的文件和文件夹
uname -m 显示机器的处理器架构
uname -r 显示正在使用的内核版本
vim
vlc
w 显示日前登入系统的用户信息
who 查看当前有哪些用户登录了本机
whoami 显示用户名称
xargs 一个给其他命令传递参数的过滤器
yes 重复输出给定的字符串直到终止它
zip 压缩工具
```
